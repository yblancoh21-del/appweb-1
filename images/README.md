Coloca aquí tus imágenes para la galería y la web.

Recomendaciones de nombres y orden:

- Para controlar el orden de la galería, usa un prefijo numérico en el nombre del archivo: `01-cover.jpg`, `02-orbital.jpg`, `03-mecha.jpg`, etc.
- Alternativamente, ajusta el atributo `data-order` en los elementos `.gallery-item` del HTML (ej. `data-order="1"`) y el script las ordenará automáticamente.
- Formatos recomendados: `.jpg`, `.png`, `.webp`.
- Tamaño sugerido: 1200px de ancho para imágenes principales, 800px para capturas.

Ejemplo de uso:

- `images/01-cover.jpg` → portada principal
- `images/02-orbital.jpg` → imagen del producto 1
- `images/05-screenshot1.jpg` → primera captura (galería)

Después de añadir/actualizar imágenes, recarga la página para ver los cambios.

Cómo usar tu logo (archivo adjunto):

- Guarda el archivo del logo que adjuntaste como `images/logo.png` dentro de este proyecto.
- Si prefieres otro formato, puedes usar `images/logo.jpg` o `images/logo.webp`, pero actualiza el `src` en los archivos HTML si cambias el nombre.
- El logo ya está referenciado en las cabeceras y enlaza al inicio. En pantallas pequeñas, sólo se mostrará el icono.

Imágenes de juegos específicas:

Para que cada juego muestre su imagen correcta, guarda las imágenes con estos nombres exactos en esta carpeta (`images/`):

**Portadas principales:**
- `baldurs-gate-3.jpg` - Portada/Hero de Baldur's Gate 3 (hero en index.html)
- `baldurs-gate-3-product.jpg` - Tarjeta de producto Baldur's Gate 3

**Productos/Tarjetas de juego:**
- `elden-ring.jpg` - Elden Ring
- `starfield.jpg` - Starfield  
- `cyberpunk-2077.jpg` - Cyberpunk 2077
- `hogwarts-legacy.jpg` - Hogwarts Legacy
- `dragons-dogma-2.jpg` - Dragon's Dogma 2

**Galería/Screenshots:**
- `game-screenshot-1.jpg` - Screenshot 1 (Planeta Aether)
- `game-screenshot-2.jpg` - Screenshot 2 (Combate)
- `game-screenshot-3.jpg` - Screenshot 3 (Estación)

Puedes descargar imágenes de:
- Steam (store.steampowered.com)
- Epic Games Store
- Sitios oficiales de los juegos
- Bases de datos de juegos (IGDB, etc.)

Tamaños recomendados:
- Portadas: 1200x600 o similar (16:9)
- Tarjetas: 400x300 o similar (4:3)
- Screenshots: 1920x1080 o similar
- Logo: 120x120 o cuadrado
